## 运行项目

### 安装依赖
```bash
cd HealthApp
npm install
```

### 启动开发服务器
```bash
npx expo start
```

### 运行应用
- 按 `i` - 在iOS模拟器运行
- 按 `a` - 在Android模拟器运行
- 扫描二维码 - 在真机运行（需要Expo Go App）

## 技术亮点

### 1. 智能营养计算
基于用户的身体信息（身高、体重、年龄、性别）和活动水平，自动计算每日所需热量和营养素目标。

### 2. AI模拟识别
模拟AI食物识别功能，展示如何通过拍照快速记录饮食，降低用户操作门槛。

### 3. 个性化建议
根据用户的健康目标（减脂/维持/增肌），提供定制化的饮食和运动建议。

### 4. 数据可视化
通过进度环、图表等可视化方式，直观展示营养摄入情况。
